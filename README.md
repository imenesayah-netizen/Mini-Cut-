# 12-Day Reset Tracker

Post-HYROX simulation recovery, debloat and rebuild tracker — Apr 26 to May 7.

**Live app:** https://imenesayah-netizen.github.io/12day-minicut

## Three phases
- **Flush** (Days 1–4): Maintenance calories, deload, anti-inflammatory focus
- **Rebuild** (Days 5–9): Light → normal training, mild deficit on rest days only
- **Peak** (Days 10–12): Full loads, slightly higher carbs, feel strong and lean

## Features
- Per-set load × reps logging for every exercise
- Auto-save to browser localStorage
- Nutrition macro tracking vs daily targets with progress bars
- Weight log with phase-coloured trend chart
- Daily rules checklist
- Live stats: days done, weight change, protein hit rate, sessions completed
