# 12-Day Mini Cut Tracker

Post-HYROX simulation mini cut tracker — Apr 26 to May 7.

**Live app:** https://imenesayah-netizen.github.io/12day-minicut

## Features
- Daily session log with per-set load × reps tracking
- Auto-save to browser local storage
- Weight log with trend chart
- Nutrition macro tracking vs targets
- Daily rules checklist
- Progress stats

## Deploy
This repo is deployed via GitHub Pages from the `main` branch root.
